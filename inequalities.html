Chapter 5: Linear Inequalities – Formula Sheet

Linear Inequality in One Variable: \( ax + b < 0, \quad ax + b \leq 0, \quad ax + b > 0, \quad ax + b \geq 0 \)
Solution Set: The set of values of \( x \) that satisfy the inequality. Represented on a number line.
Rules for Solving Inequalities:
  • Add/subtract the same number on both sides: inequality remains unchanged
  • Multiply/divide both sides by a positive number: inequality remains unchanged
  • Multiply/divide both sides by a negative number: reverse the inequality sign
Linear Inequalities in Two Variables: \( ax + by < c, \quad ax + by \leq c, \quad ax + by > c, \quad ax + by \geq c \)
Graphical Representation:
  • Convert inequality to equation: \( ax + by = c \)
  • Plot the line on the coordinate plane
  • Use test point to determine the region satisfying the inequality
  • Use dashed line for strict inequalities (< or >), solid line for inclusive (≤ or ≥)
