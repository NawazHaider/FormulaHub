Cartesian Product: \( A \times B = \{(a, b) \mid a \in A, b \in B\} \)
Relation: A relation \( R \) from set \( A \) to \( B \) is a subset of \( A \times B \)
Domain & Range: - Domain: Set of first elements in relation - Range: Set of second elements
Function: A relation in which every element of \( A \) maps to exactly one element in \( B \)
Types of Functions:
  • One-one (Injective): Different inputs map to different outputs
  • Onto (Surjective): Every element in codomain is mapped
  • Bijective: Both one-one and onto
