Basic Trigonometric Ratios: \( \sin x = \frac{\text{Perpendicular}}{\text{Hypotenuse}}, \quad \cos x = \frac{\text{Base}}{\text{Hypotenuse}}, \quad \tan x = \frac{\text{Perpendicular}}{\text{Base}} \)
Reciprocal Identities: \( \csc x = \frac{1}{\sin x}, \quad \sec x = \frac{1}{\cos x}, \quad \cot x = \frac{1}{\tan x} \)
Pythagorean Identities: \( \sin^2 x + \cos^2 x = 1 \)
\( 1 + \tan^2 x = \sec^2 x \)
\( 1 + \cot^2 x = \csc^2 x \)
Double Angle Formulas:
  • \( \sin(2x) = 2\sin x \cos x \)
  • \( \cos(2x) = \cos^2 x - \sin^2 x \)
  • \( \tan(2x) = \frac{2\tan x}{1 - \tan^2 x} \)
Periodic Properties:
  • \( \sin(x + 2\pi) = \sin x \)
  • \( \cos(x + 2\pi) = \cos x \)
  • \( \tan(x + \pi) = \tan x \)
